Averto notification generator

Averto is beautiful, light and accessible replacement for the alert() function.

The main advantage is that the averto is not modal. This means that the notification does not block the interface.

Try it yourself!

Example of this usage

This code will show same dialog, like you configured before.


Averto.show({
    timeout: 2000,
    autohide: true,
    color: '#3768F3',
    title: 'Averto',
    message: 'Beautiful and light alternative to alert()',
    blocking: false,
})

Get

Download the repository code and move dist/averto.js to the desired directory.

Or use npm:


$ npm install mishamyrt-averto --save

Setup

Link the file averto.js from the compiled sources.

If downloaded directly:


<!-- Head -->
<script src="path/to/averto.js" type="text/javascript"></script>

If installed with npm:


<script src="node_modules/mishamyrt-averto/dist/averto.js"
        type="text/javascript"></script>
